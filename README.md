#  Vehicle Insurance Data Analysis

## Project Overview
This project focuses on analyzing vehicle insurance data to understand customer behavior and identify factors that influence insurance purchase decisions.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## Dataset Information
The dataset contains information about customers such as:
- Gender
- Age
- Driving License
- Region Code
- Previously Insured
- Vehicle Age
- Vehicle Damage
- Annual Premium
- Policy Sales Channel
- Vintage
- Response

---

##  Key Steps Performed
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Understanding customer demographics  
- Identifying patterns and trends  

---

##  Key Insights
- Customer demographics play an important role in decision making  
- Previously insured customers show different behavior  
- Vehicle age and damage impact insurance purchase  

---

## Conclusion
This project helped in gaining practical experience in data analysis and understanding real-world datasets.
